# 一、題目說明

請使用 Python 製作一個具有圖形化介面（GUI）的 **台股程式交易回測系統**。

系統介面自行重新設計，風格模擬 Anaconda 內建 Designer 所設計的桌面應用介面。

使用者可輸入台灣股票代號（例如：2330），選擇 K 線週期後，按下「回測」按鈕，即可下載過去 **1 個月** 的股價資料，進行技術指標策略回測，並在視窗中顯示：

1. 最後 50 筆 K 線開高低收成交量等資料並顯示於介面表格  
2. K 線走勢圖顯示於 Label  
3. 投資績效 KPI 表格  

---

## 本題需整合：

- GUI 介面設計  
- yfinance 資料抓取  
- 技術指標計算  
- 交易策略判斷  
- 回測績效統計  
- 圖表與表格呈現

# 二、開發環境要求

請使用 Python 開發，建議環境如下：

- Python 3.x  
- GUI 套件：不限（建議 PyQt5 + Qt Designer）  
- 資料處理：pandas、numpy

# 三、使用者介面要求

需有以下輸入元件：

- 台積電(2330) 近 50 筆 30 分鐘 K 線走勢圖

# 五、交易策略需求

## 策略一：均線糾結策略

### 均線：

- MA5
- MA10
- MA20

### 糾結定義：

```python
(max(MA5, MA10, MA20) - min(MA5, MA10, MA20)) / Close <= 0.5%
```
